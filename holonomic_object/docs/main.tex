\emph{W poniższym dokumencie stosuje się notację skróconą dla funkcji trygonometrycznych, gdzie sinus zapisuje się jako $s$, a cosinus jako $c$}.\\
\\
Rozważając układy nieliniowe, których prostym przykładem jest dwuwahadło należy przeanalizować dynamikę, dzięki której opisuje się ewolucję układu dla danych warunków początkowych. Dynamikę układu bez strat energii opisuje równanie (\ref{eq:1}).

\begin{equation}
    \mathbf{M}(\ddot{q})\ddot{q} + \mathbf{C}(q, \dot{q})\dot{q} + \mathbf{D}(q) = u,\label{eq:1}
\end{equation}

gdzie

\begin{itemize}
    \item $\mathbf{M}$ to macierz bezwładności,
    \item $\mathbf{C}$ to macierz sił odśrodkowych i Coriolisa,
    \item $\mathbf{D}$ to wektor sił potencjalnych,
    \item $u$ to zadawane sterowanie,
    \item $q$ to współrzędna uogólniona.
\end{itemize}

Dla rozważanego układu macierze są zdefiniowane, jako

$$\mathbf{M} = \begin{bmatrix}
\theta_1 + 0.6\theta_4c_2 & \theta_3+0.3\theta_4c_2\\
\theta_3+0.3\theta_4c_2 & \theta_3
\end{bmatrix},$$

$$\mathbf{C} = 0.3\theta_4s_2\begin{bmatrix}
-\dot{q}_2 &-(\dot{q}_1 + \dot{q_2})\\
\dot{q}_1 & 0
\end{bmatrix},$$

$$\mathbf{D} = g\begin{bmatrix}
\theta_2c_1 + 0.4c_{12}\\
\theta_4+0.3c_{12}
\end{bmatrix},$$

gdzie $\theta_i$ to parametry inercyjne, równe odpowiednio

\begin{itemize}
    \item $\theta_1 = I_1 + I_2 + m_2l_1^2~[\text{kgm}^2]$,
    \item $\theta_2 = m_1p_1 + m_2l_1=9.5~[\text{kgm}]$,
    \item $\theta_3 = I_2 = 0.24~[\text{kgm}^2]$,
    \item $\theta_4 = m_2p_2 = 0.77~[\text{kgm}]$,
\end{itemize}

gdzie $I_i$ to moment bezwładności, $m_i$ masa, $l_i$ długość, $p_i$ $i$-tego ramienia.

Macierz $\mathbf{M}$ posiada charakterystyczne cechy, które ją definiują. Są to:

\begin{enumerate}
    \item Symetryczność $\mathbf{M}^T = \mathbf{M}$,
    \item Dodatniaokreśloność $\mathbf{M} > 0$,
    \item Skośna symetria $\dot{\mathbf{M}} = \mathbf{C} + \mathbf{C}^T$
\end{enumerate}

    \item algorytm QuDorsey'a nie daje gwarancji zbieżności
    \item człon proporcjonalny odpowiada za samą regulację, jako wprowadzenie oscylacji wokół położenia równowagi
    \item człon różniczkujący odpowiada za gładkość stabilizacji
    \item im większy człon proporcjonalny, tym szybsza stabilizacja i większe oscylacje
    \item im większy człon różniczkujący, tym wolniejsza stabilizacja, ale gładsza
    \item algorytm dokładnej linearyzacji gwarantuje zbieżność
