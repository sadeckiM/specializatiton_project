# Projekt specjalnościowy 2026
## Michał Sadecki
### AiR 2023, specjalność Robotyka

Projekt składa się z dwóch części: holonomicznej i nieholonomicznej. Projekt
jest realizowany w środowisku MATLAB/Simulink, gdzie po zaimplementowaniu odpowiednich
modeli matematycznych układów przeprowadzane są badania symulacyjne dla zadanych
parametrów.

Część holonomiczna składa się z implementacji algorytmu QuDorsey'a oraz
dokładnej linearyzacji. Całość jest podsumowana raportem (w.i.p).
